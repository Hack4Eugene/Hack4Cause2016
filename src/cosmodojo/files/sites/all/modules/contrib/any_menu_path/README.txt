README.txt
==========

-- SUMMARY --

This module allows the user to add menu paths that don't exist on the site.

-- REQUIREMENTS --

None.

-- INSTALLATION --

* Install as usual, see http://drupal.org/node/70151 for further information.

-- USAGE --

Every time you add a new link to a menu
(i.e.: admin/structure/menu/manage/main-menu/add)
a new checkbox will appear at the top: 'No
Path Validation'. Simply enable it and this
module will do its magic.

You can view all overriden paths at:
admin/config/content/any_menu_path or by visiting 
Configuration -> Any Menu Path Override Listing
